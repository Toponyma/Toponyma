# Dataset: Proměny gazelu v české poezii

[cite_start]Tento repozitář obsahuje datový soubor 253 básní (českých gazelů), který tvoří materiálovou základnu pro studii *Ve tvých rýmech duše má se kochá… Proměny gazelu v české poezii*[cite: 1, 259].

## O datech
Tento kvantitativní výzkum sleduje, jak se původně perská forma chová v odlišném jazykovém a kulturním prostředí střední Evropy během 19. a první poloviny 20. století. Korpus byl sestaven primárně s využitím infrastruktury Česká literární bibliografie (korpus PoeTree.cs) a doplněn o ručně excerpované časopisecké publikace.
